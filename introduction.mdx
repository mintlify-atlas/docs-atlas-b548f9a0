## Welcome to AnyCarousel

AnyCarousel is a powerful, highly adaptable carousel component that covers real use cases like custom content, snapping, auto scroll, and responsive layout. It works for all kinds of content—images, videos, and custom components.

## Key Features

## Why AnyCarousel?

<Accordion title="Designed for Real Use Cases">
  Unlike generic carousel libraries, AnyCarousel is built to handle real-world requirements: custom content types, precise control over animations, mobile-first touch interactions, and accessibility out of the box.
</Accordion>

<Accordion title="Developer-Friendly API">
  Simple, intuitive props with TypeScript support. Get started in minutes, customize everything when you need to.
</Accordion>

<Accordion title="Performance-Optimized">
  Lightweight bundle size, efficient rendering, and smooth 60fps animations using native browser APIs.
</Accordion>

<Accordion title="Fully Customizable">
  From easing curves to navigation icons, every aspect of the carousel can be tailored to match your design system.
</Accordion>

## Quick Example

```tsx
import { Carousel } from "react-any-carousel";

function App() {
  return (
    <Carousel 
      scrollSnapType="center" 
      scrollOffset={300} 
      duration={600}
    >
      <div className="card">Item 1</div>
      <div className="card">Item 2</div>
      <div className="card">Item 3</div>
    </Carousel>
  );
}
```
